# 🎮 C-Python Hybrid Janken System

C言語で作られた超軽量な本体エンジンから、最新のPython 3.13環境をバックエンドとして呼び出して動かす、ハイブリッドなじゃんけんゲームです。

C言語の「圧倒的な軽さ（133.16KiB）」と、Pythonの「高度なロジック処理（動的配列・辞書機能）」の適材適所を体現したシステム設計になっています。

## 🚀 特徴
- **超軽量バイナリ**: メモリ管理を手動で行い、無駄な機能を削ぎ落とすことで、実行ファイル（`main.exe`）のサイズをわずか **133.16KiB** に抑えています（自作Go言語ツール `my_explorer` にて検証済み）。
- **洗練された勝敗判定数式**: 冗長な条件分岐（if文の連打）を排除し、`(user_hand - ai_te + 3) % 3` という数式による美しい数学的ジャッジを実装。
- **徹底したメモリ救出 (`save()`)**: `Py_DECREF` を適切に配置し、8GB環境のパソコン（ThinkPad L13）をメモリリークによる「熱暴走（焼き鳥状態）」から完全に守る安全設計。

## 📁 構成ファイル
- `main.c`: ユーザー入力を受け付けるC言語の無限ループエンジン。Pythonの埋め込み（Embedding）を担当。
- `main.py`: `random`モジュールを使用したAIの手の生成、および勝敗の判定ロジック。
- `README.md`: この説明書。

## 🛠️ ビルドと実行方法

Windows環境にて `w64devkit`（GCC環境）を使用して手動でビルドします。

### 1. コンパイル（一撃コマンド）
`w64devkit` のターミナル（`sh`）を開き、以下のコマンドを実行してCコードをネイティブな機械語の実行ファイル（`.exe`）に自動翻訳します。
※Pythonのインストールパスは、お使いの環境（Python 3.13）に合わせて適宜書き換えてください。

```bash
gcc -O2 -msse3 -I"C:/Users/Owner/AppData/Local/Programs/Python/Python313/include" main.c -L"C:/Users/Owner/AppData/Local/Programs/Python/Python313/libs" -lpython313 -o main.exe
```

### 2. バイナリの極限最適化（オプション）
不要なデバッグ情報を削ぎ落とし、ファイルサイズを極限まで軽量化（100KB前後）します。

```bash
strip main.exe
```

### 3. 実行
同じフォルダに `main.py` が配置されていることを確認し、以下のコマンドでシステムを起動します。

```bash
./main.exe
```

## 🎮 遊び方
起動後、ターミナル上で以下の数字を入力してエンターを押してください。

- `0`: 👊 グー
- `1`: ✌️ チョキ
- `2`: ✋ パー
- `-1`: ❌ ゲームを安全に終了（メモリをすべて解放してクローズ）
